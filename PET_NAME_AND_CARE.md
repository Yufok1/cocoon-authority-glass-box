# Mira-Kite Cocoon

Mira is the association/meaning caretaker label for organism `16525dfc6c33419b`.
Kite is the action/reasoning caretaker label for organism `0c019d24a56b0086`.

The original IDs, alliance/trust records, fitness, vocabulary, atomic language state, knowledge web, conversation traces, and training logs are preserved inside `cocoon_cognition_agency.py`.

Cognitive action-head labels trained in the second pass:
- 0 observe
- 1 associate
- 2 recall
- 3 reason
- 4 respond
- 5 verify

Training method: fixed points first, then semantic bridges, then short reasoning ladders, then action-head faculty routing.

## Local Trainer App

Run the phone/browser trainer with:

```bash
python mira_kite_authority.py --cocoon cocoon_cognition_agency.py --host 127.0.0.1 --port 8765
```

Open `http://127.0.0.1:8765/` on the Android browser.
